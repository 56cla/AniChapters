## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by opening a private issue on GitHub or contacting the maintainer.

Please include as much detail as possible, such as:
- A description of the vulnerability
- Steps to reproduce the issue
- Any potential impact

You can expect a response within a few days. If the vulnerability is confirmed, it will be fixed as soon as possible and a security update will be released.
