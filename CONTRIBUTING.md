# Contributing

Thank you for considering contributing to this project!

## How to Contribute

- Fork the repository
- Create a new branch for your changes
- Make your changes and commit them
- Open a Pull Request describing what you changed

## Guidelines

- Keep changes focused and clear
- Do not break existing functionality
- Follow the existing code style
- Test your changes before submitting

## Reporting Issues

If you find a bug or have a feature request, please open an issue on GitHub and include as much detail as possible.

## Suggestions

Ideas, improvements, and feedback are always welcome.
